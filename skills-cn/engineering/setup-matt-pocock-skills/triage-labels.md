# Triage 标签

这些 skill 使用五种规范的 triage 角色。本文件将这些角色映射到本仓库 issue 跟踪器中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本跟踪器中的标签 | 含义 |
| -------------------------- | ---------------- | ---- |
| `needs-triage`             | `needs-triage`   | 维护者需要评估该 issue |
| `needs-info`               | `needs-info`     | 等待报告者提供更多信息 |
| `ready-for-agent`          | `ready-for-agent`| 已完整说明，可由离线 Agent 处理 |
| `ready-for-human`          | `ready-for-human`| 需要人工实现 |
| `wontfix`                  | `wontfix`        | 不会处理 |

当 skill 提到某个角色（如“应用已准备好供离线 Agent 处理的 triage 标签”）时，使用此表中对应的标签字符串。

请编辑右侧列，使之与实际使用的词汇一致。
